# Ethereum Swap

Swap ETH for BTC

## Getting Started
```
$ npm install
$ npm install -g webpack webpack-cli
$ npm run start
```



## Using the Ethereum bridge for local Testing
```
npm install -g ethereum-bridge
ethereum-bridge -a 9 -H 127.0.0.1 -p 7545 --dev
```
more (info)[https://github.com/oraclize/ethereum-bridge]
